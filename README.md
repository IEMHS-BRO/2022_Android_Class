## 목표
### 학습 목표
- 다양한 프로젝트를 개발해보며 안드로이드 앱 개발에 필요한 기본 지식 학습
- 실무에서 사용하는 툴과 방식을 경험해보며 실무적인 지식 학습
- 다른 개발자와 커뮤니케이션을 통한 협업 경험 학습

### 최종 목표
- RESTful API 연동을 통한 중고 플랫폼 앱 만들기
